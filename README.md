# شرح الفكرة
المشكلة والحلول المطروحة إن وجدت
فكرة المشروع هي التبرع باي شي موجود والمميز في البرنامج ان المتبرع لن يضطر للذهاب الى اللجان الخيرية بنفسه مجرد ان يزودنا بمعلوماته ونوع المنتج الذي يود التبرع فيه ونحن سنصل اليه لنأخذ تبرعاته سيكون هناك مكافآت للمتبرعين لتشجيعهم على التبرع 
  

# الأعمال المستقبلية


الاعمال المستقبلية هي الاشتراك مع اكثر من لجان خيرية، والتعامل مع المطاعم لتزويدنا ببقايا طعامهم بدل ان يرمى والتبرع فيه،
  ايضا نود ان نصل الى اكبر عدد ممكن من المتبرعين ونزود المناطق بالثلاجات لحفظ الاكل و صناديق واماكن للتبرع بمستلزمات المنزل وغيره وايضا الملابس
